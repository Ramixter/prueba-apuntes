    Direcciones IP (IPv4 e IPv6)¶

    Nota

    Las direcciones IP son identificadores numéricos únicos que se utilizan para identificar dispositivos en una red, como ordenadores, routers, servidores y otros dispositivos conectados a Internet.

    Existen dos versiones de direcciones IP: IPv4 e IPv6. La versión IPv4 utiliza un formato de dirección de 32 bits y se utiliza actualmente en la mayoría de las redes. La versión IPv6 utiliza un formato de dirección de 128 bits y se está implementando gradualmente en todo el mundo para hacer frente a la escasez de direcciones IPv4.

    Las direcciones IPv4 se representan como cuatro números separados por puntos, como 192.168.0.1, mientras que las direcciones IPv6 se representan en notación hexadecimal y se separan por dos puntos, como 2001:0db8:85a3:0000:0000:8a2e:0370:7334.

    Una dirección IP la tenemos que ver como una etiqueta numérica que indentifica de manera lógica y jerarquica a una interfaz en la red de un dispositivo que utilice el protocolo de internet.

    Podemos usar el comando ifconfig para ver la ip de nuestra máquina:

    Ejemplo 

    ifconfig

    enp0s3: flags=4163<UP,BROADCAST,RUNNING,MULTICAST>  mtu 1500
    inet 192.168.1.132  netmask 255.255.255.0  broadcast 192.168.1.255
    inet6 fe80::b79e:5c10:c25c:8701  prefixlen 64  scopeid 0x20<link>
    ether 08:00:27:2d:73:8e  txqueuelen 1000  (Ethernet)
    RX packets 21  bytes 2069 (2.0 KiB)
    RX errors 0  dropped 8  overruns 0  frame 0
    TX packets 21  bytes 3364 (3.2 KiB)
    TX errors 0  dropped 0 overruns 0  carrier 0  collisions 0

    Si por ejemplo hicieramos un hostname -I veríamos que nos reporta únicamente cual es nuestra IP:

    Ejemplo 

    hostname -I

    192.168.1.132

    Tenemos que recordar que 1 byte son 8 bits, por lo que a la hora de tratar de representar la IP con un total de 32 bits para poder representar nuestra IP:

    • Sabemos que nuestra IP es la siguiente: 192.168.1.132, por lo que podemos hacer lo siguiente para representar nuestra IP en los octetos separados:

      echo "192.168.11.42"

      192.168.11.42


      echo "$(echo "obase=2; 192" | bc).168.11.42"

      11000000.168.11.42


      echo "$(echo "obase=2; 192" | bc).$(echo "obase=2; 168" | bc).$(echo "obase=2; 11" | bc).$(echo "obase=2; 42" | bc)"

      11000000.10101000.1011.101010
